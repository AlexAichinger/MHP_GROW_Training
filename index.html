Task Section

Task 1 - Getting started

Fork this codesandbox, so that you have your own one to work in. As a first step, please implement the methods orderByAlphabet and filterAllToxicFruits. You can use whatever sorting algorithm you want. After you completed your implementation the corresponding unit tests in fruits.spec.ts should be passing.

Task 2 - TDD

Please implement the Unit Test "Testing that fruits are grouped correctly". You can use the following snippet as an starting point

test("3 is 3", () => {
        expect(3).toBe(3);
      });
After your Unit Test implementation is done, please implement the actual grouping logic and make sure that the Unit Test passes at the end.

Task 3 - Fruit Salad Generator

Implement a fruit salad generator that fullfills the following criteria:

  • Minimum three different tastes included
  • No toxic ingredients
  • Minimum five different fruits
  • No taste more than twice
Bonus: Implement a Unit Test that proves your work.
